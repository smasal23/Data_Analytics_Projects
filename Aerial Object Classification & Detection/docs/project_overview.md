# Project Overview

This project focuses on aerial visual intelligence across two tasks:

1. **Binary Classification**  
   Determine whether an input image belongs to class:
   - Bird
   - Drone

2. **Object Detection**  
   Detect aerial objects in images using YOLO-style annotations.

The project is designed around a modular architecture with Colab + Drive persistence, making it suitable for experimentation, training, reporting, and later deployment.
